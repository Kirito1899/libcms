Язык запросов

Термы

Запросы состоят из термов и операторов.
Термы бывают нескольких типов. Первые из них, это обычные слова, например, «банк», «отчет». Второй же тип термов это группа слов, например, «центральный банк». Несколько термов можно связывать вместе при помощи логических операторов.

Поля

Поиск данных в заданном поле записи. Для поиска по определенному полю необходимо в запросе написать название этого поля, дальше поставить двоеточие и после него написать терм. Например:

title_t:"центральный банк"

Если при запросе в атрибуте используется несколько слов, то их необходимо заключать в кавычки или в скобки

title_t:центральный
title_t:"центральный банк"
title_t:(центральный AND банк AND россии)
title_t:(центральный AND банк AND россии OR ЦБ)

Wildcards

Поиск записей по маске. Можно заменит один ? или несколько * символов например:

title_t:бан?
title_t:цент*й

Нечеткий поиск

Для написания такого типа запросов необходимо поставить символ ~ в конце одиночного терма, например, запрос 

title_t:(централый~ бнк~ росии~)

найдет записи с заглавием, похожи на центральный банк россии, хотя при запросе была допущена ошибка.
Можно задать коэффициент похожести:

title_t:отчет~0.9

По-умолчанию это значение равно 0.5.
Для термов из нескольких слов символ ~ используется для указания максимального расстояния между ними.

Усиление термов

Чтобы записи со словом «ЦБ» были более релевантными нежели записи со словом «банк», тогда можно построить запрос следующим образом:

title_t:(ЦБ^5 OR банк)

Тоже самое можно делать и для фраз. По-умолчанию значение этой характеристики равно 1 и оно должно быть положительным (таким образом можно задать значение равное 0.1, например).

Логические операции

Логические операции позволяют комбинировать несколько термов. Например:

title_t:банк OR ЦБ

или

title_t:(банк OR цб) AND subject_keywords_t:отчет

Группировка

Логические выражения можно группировать:

(банк OR цб) AND отчет

Экранирование символов

Символы, которые необходимо экранировать при помощи символа «\»:

+ - && || ! ( ) { } [ ] ^ " ~ * ? : \
